================================================================================
              WHERE TO FIND VOICE CLONING IN THE WEB INTERFACE
================================================================================

STEP 1: Open your web browser and go to:
        → http://localhost:5003

STEP 2: You will see the TTS Voice Generator page

STEP 3: Look for the "TTS Engine" section (near the top of the page)
        You will see THREE radio button options:

        ┌────────────────────────────────────────────────────────────┐
        │  TTS Engine                                                │
        │                                                            │
        │  ○  Piper TTS                                              │
        │     Wide variety of voices and languages                   │
        │                                                            │
        │  ○  Silero TTS                                             │
        │     High-quality neural voices with GPU acceleration       │
        │                                                            │
        │  ○  Chatterbox TTS    ← ← ← CLICK THIS ONE! ← ← ←        │
        │     AI voice cloning - Clone any voice!                    │
        │                                                            │
        └────────────────────────────────────────────────────────────┘

STEP 4: Click the "Chatterbox TTS" option (the 3rd one)

STEP 5: The page will show a NEW SECTION called "Voice Cloning"
        with:
        - File upload box
        - Language selector
        - Test Voice button
        - Save to Library button
        - Your Cloned Voices library

STEP 6: Upload a 3-10 second audio file and start cloning!

IF YOU DON'T SEE THE "CHATTERBOX TTS" OPTION:

1. Hard refresh the browser page:
   - Windows: Press Ctrl + F5
   - Mac: Press Cmd + Shift + R

2. Clear browser cache and reload

3. Try opening in a different browser

4. Verify container is running:
   docker-compose ps tts-web-interface
   (Should show "Up" and "healthy")
